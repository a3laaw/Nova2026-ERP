# **App Name**: NovaFlow ERP

## Core Features:

- Integrated Operation-to-Finance Pipeline: Automatically generates accounting journal entries from operational events like contract activations, inventory receipts (GRNs), and payroll approvals.
- WBS & GPS Field Visits: Project management using Work Breakdown Structure (WBS) and Bill of Quantities (BOQ), with GPS-linked field visit reporting for site engineers.
- Smart Procurement Logic: Full lifecycle tracking from Purchase Request to RFQ comparison, Purchase Order issuance, and warehouse receipt with automated cost calculation.
- Labor-Law Compliant HR: Automated payroll generator featuring a sick-leave ladder and indemnity (EOSB) calculator specifically tailored for Kuwaiti and Gulf labor laws.
- AI Decision Support Tool: A generative AI tool that analyzes supplier quotes for comparison and provides cash flow projections based on project milestones.
- Multi-Tenant Firestore Ledger: Data-isolated architecture ensuring secure, real-time financial data storage and client-specific chart of accounts.
- Executive Financial Reporting: Real-time generation of Trial Balances, P&L statements, and Balance Sheets using verified entries from the operational modules.

## Style Guidelines:

- The palette is driven by the 'Executive Energy' concept, featuring a primary Burnt Orange (#e87c24) for action, set against a warm ivory background (#fdfaf3) for a high-end, comfortable user experience.
- A deep navy accent (#1e1b4b) is used for headers to provide contrast and a sense of institutional stability, paired with a vibrant accent (#E82436) for critical alerts.
- Headlines feature 'Space Grotesk' for a precise, techy engineering feel. 'PT Sans' is used for body text and financial tables to ensure modern readability across complex data sets.
- Line-art Lucide icons with subtle gradients for navigation items, emphasizing architectural and engineering themes.
- A clean dashboard-centric layout using oversized cards with 2xl border-rounding and glassmorphic navigation bars.
- Smooth layout transitions between dashboard modules and subtle loading state pulses for real-time Firestore updates.
