
# سجل تطوير مشروع NovaFlow ERP 🚀

هذا الملف هو المرجع الرسمي لكل التطورات والتعديلات التي تتم في النظام.

## [1.1.0] - بناء هيكل الـ SaaS المتقدم
### ✅ معمارية الدخول والتسجيل (SaaS Onboarding)
- **تحويل التسجيل لطلبات:** لم يعد التسجيل ينشئ حساباً فورياً، بل يرسل طلباً إلى `company_requests`.
- **لوحة المطور (`/developer`):** إنشاء لوحة تحكم خاصة للمطور لمراجعة وتفعيل طلبات الشركات الجديدة.
- **التوجيه بناءً على الدور:** تعديل منطق الدخول ليتم توجيه المطور إلى `/developer` وباقي المستخدمين إلى `/dashboard`.
- **حماية المسارات:** إضافة `DeveloperLayout` لمنع غير المخولين من دخول أدوات المطور.

### ✅ تحديث البيانات (Global Index)
- تحديث `backend.json` ليشمل كيان `CompanyRequest` وخصائص `isDeveloper` و `username` في `GlobalUser`.
- ربط صفحة الدخول بالـ `GlobalUser` لضمان صحة التوجيه.

## [1.0.0] - التأسيس والربط (Infrastructure)
- ربط المشروع بـ Firebase (Auth & Firestore).
- إعداد نظام Multi-tenancy (عزل بيانات كل شركة عن الأخرى).
- تفعيل قواعد الحماية (Security Rules) لضمان خصوصية البيانات.
- إنشاء نظام مركزي لمعالجة الأخطاء (FirebaseErrorListener).
- تحويل النظام ليكون ثنائي اللغة (عربي/إنجليزي) بالكامل.
- دعم اتجاهات الصفحة (RTL/LTR) تلقائياً عند التبديل.
- إعداد نظام التصميم باستخدام Tailwind CSS و ShadCN UI.
- **وحدة CRM:** شاشة إدارة العملاء والفرص مع ربط حي بقاعدة البيانات (CRUD).
- **وحدة المحاسبة:** مساعد ذكي لمطابقة القيود البنكية.
- **وحدة الموارد البشرية:** نظام فحص المستندات بالذكاء الاصطناعي.
- **وحدة المشاريع:** نظام التنبؤ بالتدفقات النقدية (Cash Flow Forecasting).
