
# سجل تطوير مشروع NovaFlow ERP 🚀

## [1.2.9] - أتمتة تفعيل المنشآت (SaaS Provisioning)
### ✅ التحديثات (Updates)
- **التفعيل الذكي:** تحديث زر "تفعيل" ليقوم بإنشاء سجل المنشأة (Company) تلقائياً في Firestore بمجرد القبول.
- **سياسة الاشتراك:** تطبيق القواعد التلقائية: (14 يوماً فترة تجريبية + حد أقصى 5 مستخدمين) لكل منشأة جديدة.
- **تحسين المعمارية:** استخدام `writeBatch` لضمان تحديث حالة الطلب وإنشاء الشركة في عملية واحدة ذرية (Atomic Operation).
- **تحديث البيانات:** إضافة حقول `trialEndsAt` و `maxUsers` إلى مخطط كيان الشركة في `backend.json`.
