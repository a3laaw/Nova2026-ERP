# سجل تطوير مشروع NovaFlow ERP 🚀

## [1.9.0] - نظام الموارد البشرية والرواتب المتكامل (HR & Payroll Engine)
### ✅ التحديثات التقنية (HR Core v3)
- **إدارة الإجازات الذكية**: تنفيذ محرك حساب أيام العمل الفعلية باستبعاد العطلات الرسمية والأسبوعية، مع كشف التداخل الزمني آلياً.
- **شرائح الإجازات المرضية**: دمج منطق قانون العمل الكويتي لتقسيم راتب الإجازة المرضية (كامل، 75%، 50%، 25%، بدون راتب).
- **نظام الاستئذانات المطور**: تقييد الاستئذانات بحد أقصى 3 ساعات للطلب و12 ساعة شهرياً، مع فحص التداخل مع الإجازات.
- **محرك البصمة والاستيراد**: دعم رفع ملفات CSV للحضور وربطها بالموظفين مع حساب دقائق التأخير والانصراف المبكر لحظياً.
- **توليد الرواتب الذكي**: نظام يدمج (الراتب الأساسي، البدلات، غيابات الحضور، استثناءات الاستئذان) لتوليد مسودات رواتب دقيقة.
- **دورة اعتماد الرواتب**: إضافة حالات (Draft -> Reviewed -> Approved -> Paid) مع توثيق المستخدم والوقت لكل إجراء.
- **التكامل المحاسبي (Accounting Hook)**: توليد آلي لبيانات قيود اليومية عند تسجيل صرف الرواتب.
- **حاسبة مكافأة نهاية الخدمة**: محرك قانوني يحسب المستحقات بناءً على سنوات الخدمة، سبب التركه (استقالة/إنهاء)، وبدل الإجازات والإنذار.
- **مركز التقارير التحليلي**: إضافة تقارير الحضور، الرواتب، أرصدة الإجازات، والملف الشامل للموظف (Employee Dossier) القابل للطباعة.
- **قواعد الأمان (Firestore Rules)**: تنفيذ حماية Multi-tenant مع وصول مستند للأدوار (RBAC) وحماية سجلات التدقيق من التعديل.
