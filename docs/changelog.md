
# سجل تطوير مشروع NovaFlow ERP 🚀

## [1.3.8] - تطوير موديول القوائم المرجعية (Reference DB Upgrade)
### ✅ التحديثات (Updates)
- **إدارة البنود:** إضافة ميزة إضافة وحذف بنود التحقق (Checklist Items) لحظياً قبل حفظ القائمة المرجعية.
- **الامتثال للـ UI Policy:** 
    - إزالة كافة القيم الافتراضية من حقول الإدخال.
    - تحسين الـ Placeholders لتكون توضيحية ومحفزة للإدخال (مثال: "إجراءات استلام حديد التسليح").
    - ضمان إخفاء أسهم التحكم في حقول الأرقام (عالمياً).
- **تحسين الواجهة:** إضافة بطاقات إحصائية جديدة وتحديث جدول عرض القوائم لدعم عرض عدد البنود المضافة.

## [1.3.7] - تطبيق سياسة UI/UX الموحدة (UI Policy Patch)
### ✅ التحديثات (Updates)
- **إخفاء أسهم الأرقام:** إضافة قاعدة CSS عالمية لإخفاء أزرار الزيادة والنقصان (Spinners) من حقول الأرقام لضمان واجهة نظيفة.
- **إزالة القيم الافتراضية:** تم تفريغ كافة حقول الإدخال من القيم الافتراضية (مثل 0 أو 5) واستبدالها بخاصية `placeholder`.
- **تحسين الـ Placeholders:** استخدام نصوص توضيحية داخل الحقول تختفي فورياً عند النقر أو الكتابة لتحسين تجربة المستخدم.
- **إصلاح استقرار الواجهة:** معالجة مشكلة تحول الحقول من "Controlled" إلى "Uncontrolled" عبر ضمان وجود قيم نصية دائماً.
