
# سجل تطوير مشروع NovaFlow ERP 🚀

## [1.3.0] - نظام التفعيل الذاتي (Self-Service SaaS)
### ✅ التحديثات (Updates)
- **التفعيل التلقائي:** تحويل صفحة التسجيل `/register` لتقوم بإنشاء المنشأة (Company) وحساب المستخدم (Global User) فوراً دون الحاجة لموافقة المطور.
- **إدارة الهوية:** إضافة حقول "اسم المستخدم" و "كلمة المرور" في نموذج التسجيل لتمكين الدخول الفوري.
- **سياسة التفعيل الآلي:** تطبيق قاعدة (14 يوماً تجريبية + 5 مستخدمين) برمجياً عند إنشاء السجل.
- **تحسين تجربة المستخدم (UX):** إضافة ميزة "إظهار/إخفاء كلمة المرور" وتوجيه المستخدم تلقائياً للوحة التحكم بعد النجاح.
- **تحديث البنية التحتية:** استخدام `writeBatch` لضمان تكامل البيانات بين ملفات الشركة والمستخدم العالمي في عملية واحدة.
